# Antlr Multi-language template

### Javascript
```
npm install
make jst
```

### Java

```
gradle wrapper --gradle-version 3.4
make javat
```

### Python

If you're not already running python 3:
```
virtualenv --python=python3 pyenv
. ./pyenv/bin/activate
```

```
pip install antlr4-python3-runtime
make pyt
```
